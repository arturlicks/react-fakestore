# React Fake Store

An e-commerce style product catalog built with **Next.js 16 (App Router)** and **TypeScript**, consuming a public REST API to render and browse product listings.

Built as a hands-on exercise with the modern Next.js App Router data-fetching model, typed components, and Tailwind CSS v4.

🔗 **Live demo:** _add your Vercel deployment link here_

## Features

- Product catalog rendered via the Next.js App Router
- Fully typed components and API responses (TypeScript)
- Styled with Tailwind CSS v4

## Tech Stack

| Layer | Technology |

## Getting Started

### Prerequisites

- Node.js 18+

### Installation

```bash
git clone https://github.com/arturlicks/react-fakestore.git
cd react-fakestore
npm install
```

### Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app. The page auto-updates as you edit `app/page.tsx`.

### Build for production

```bash
npm run build
npm start
```
